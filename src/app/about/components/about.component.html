Request App

Request App is an implementation of a data gathering interface that presents only relevant and required options. Relevance is established based on previous responses. Reducing number of available options lowers pereceived complexity of request completion.

This approach shares guidance aspects of wizzard interface pattern with an addition of maintaining context of previously provided responses that are context switch out in wizzard interface.

Request is summarized in hierarhical text format and used in email notifications

The demo app is based on help desk request entry scenario where depending on intial request significantly different set of responses can be available or required.
